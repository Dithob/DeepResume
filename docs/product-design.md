# Product Design

CV2Offer Agent helps job seekers turn the gap between their current resume and a target JD into a concrete learning loop.

## MVP Workflow

```text
Resume + JD
  -> structured parsing
  -> gap report
  -> learning plan
  -> task progress
  -> career evidence
```

## MVP Views

- Import: paste resume text, JD text, or Markdown learning plan.
- Gap: show match score, strengths, missing core skills, weak-evidence skills, and learning priorities.
- Plan: show weekly milestones and task status.
- Evidence: show completed task outputs, artifact links, strength, and resume bullet drafts.
