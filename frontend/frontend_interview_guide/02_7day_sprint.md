# 7 天短期冲刺复习

## 使用原则

这份计划适合面试前 7 天使用。每天分成四块：

- 知识复习：当天必须重新过一遍的概念。
- 手撕练习：当天算法题或前端手写方向。
- 项目输出：把知识点和你的简历项目绑定。
- 自测问题：晚上用来模拟面试。

每天建议 6-8 小时。如果时间只有 3-4 小时，优先级是：项目深挖 > JS/浏览器基础 > 手写题 > 框架八股。

## Day 1：JavaScript、TypeScript 与手写题

### 知识复习

- 作用域、闭包、原型链、`this` 绑定。
- Promise、async/await、宏任务、微任务、事件循环。
- TypeScript 类型收窄、泛型、接口、工具类型。
- 模块化、Tree Shaking、ESM 与 CommonJS 差异。

### 手撕练习

推荐题型：

- 防抖、节流、深拷贝。
- `Promise.all`、并发请求限制。
- 发布订阅、事件委托。
- 数组扁平化、数组转树。

解题要点：

- 先说 API 语义，再写代码。
- 异步题必须说清 resolve/reject、顺序和错误传播。
- 深拷贝要说明循环引用、Date、RegExp、函数如何处理。

### 项目输出

把基础知识绑定到智能应用生成平台：

- SSE 消息增量渲染本质是异步事件流，和 Promise 一次性完成不同。
- DOM 监听脚本注入要理解事件捕获/冒泡、事件委托和解绑。
- `postMessage` 要处理消息类型、来源校验和异常消息。

### 自测问题

- 微任务和宏任务的执行顺序是什么？
- `async/await` 的异常如何捕获？
- TypeScript 泛型在组件封装里有什么价值？
- 你的可视化编辑为什么适合用事件委托？

## Day 2：CSS、布局与响应式

### 知识复习

- 盒模型、BFC、层叠上下文。
- Flex、Grid、定位、响应式布局。
- `rem`、`vw/vh`、媒体查询、容器宽度约束。
- `transform`、`opacity`、重排、重绘、合成。

### 手撕练习

推荐题型：

- 水平垂直居中。
- 两栏/三栏布局。
- 商品卡片自适应网格。
- 文字溢出省略、多行省略。
- 移动端预览容器等比例缩放。

### 项目输出

把 CSS 绑定到校园二手交易平台和智能应用生成平台：

- 商品列表如何用 Grid/Flex 适配不同宽度。
- `iframe` 预览站点如何做桌面/移动端切换。
- 表单、弹窗、卡片如何抽象成稳定组件。

### 自测问题

- BFC 是什么，能解决哪些布局问题？
- Flex 和 Grid 分别适合什么场景？
- 如何避免弹窗或按钮文字在移动端溢出？
- `transform` 为什么通常比修改 `top/left` 更适合动画？

## Day 3：Vue3、Router、Pinia 与组件通信

### 知识复习

- Vue3 响应式：`ref`、`reactive`、`computed`、`watch`。
- 生命周期、Composition API、可复用 hooks。
- `props`、`emits`、插槽、`provide/inject`。
- Vue Router 导航守卫、动态路由、权限控制。
- Pinia/Vuex 状态管理、异步 action、持久化。

### 手撕练习

推荐题型：

- 写一个 `useTaskPolling`。
- 写一个 `useEventSource`。
- 写一个简单表单组件，支持 `v-model`。
- 写一个路由权限判断函数。

### 项目输出

准备云端图像协同系统答案：

- 路由 `meta.access` 如何设计？
- 登录态在哪里维护？
- 权限失效如何处理？
- 图表组件如何封装和销毁？

### 自测问题

- `computed` 和 `watch` 的区别是什么？
- Vue3 响应式为什么不能直接解构 `reactive` 对象？
- 路由守卫中异步获取用户信息如何避免死循环？
- Pinia 和本地缓存如何配合？

## Day 4：浏览器、网络、安全与实时通信

### 知识复习

- HTTP/HTTPS、状态码、缓存、跨域、CORS。
- Cookie、Token、localStorage、sessionStorage。
- 浏览器渲染流程、事件传播、同源策略。
- SSE、WebSocket、轮询的区别。
- XSS、CSRF、点击劫持、CSP、`iframe sandbox`。

### 手撕练习

推荐题型：

- 封装 `fetch` 请求，包含超时和错误处理。
- 设计一个 WebSocket 重连策略。
- 实现简化版事件总线。
- 实现请求并发控制。

### 项目输出

准备两个链路：

- 智能应用生成平台：SSE 流式接口 -> 打字机输出 -> 结束/错误状态。
- 云端图像协同系统：WebSocket 连接 -> 编辑锁 -> 冲突提示 -> 重连恢复。

### 自测问题

- SSE 和 WebSocket 的核心差异是什么？
- `postMessage` 为什么必须校验 `origin`？
- Token 存在 localStorage 有什么风险？
- 如果 WebSocket 断线，前端如何恢复状态？

## Day 5：性能优化、工程化与测试

### 知识复习

- Core Web Vitals：LCP、INP、CLS。
- 首屏优化、路由懒加载、图片优化、缓存策略。
- Vite 构建、代码分割、按需引入、产物分析。
- ESLint/Prettier、Git、接口 Mock。
- Vitest、Playwright、组件测试和 E2E 测试。

### 手撕练习

推荐题型：

- 虚拟列表核心思路。
- 图片懒加载。
- 防抖搜索。
- 组件缓存或结果缓存。

### 项目输出

把性能和工程化绑定到项目：

- ECharts 大数据量如何优化？
- AI 对话流式输出如何避免频繁渲染卡顿？
- `iframe` 预览如何隔离错误？
- 如何用 JMeter 或浏览器 Performance 面板定位慢请求/卡顿？

### 自测问题

- LCP、INP、CLS 分别反映什么体验？
- 路由懒加载为什么能优化首屏？
- 如何判断卡顿来自 JS 执行、布局还是网络？
- 前端测试最适合覆盖哪些稳定逻辑？

## Day 6：项目深挖与算法题

### 知识复习

- 云端智能图像协同系统：权限、异步任务、WebSocket、冲突检测、ECharts。
- 智能应用生成平台：SSE、`iframe`、`postMessage`、DOM 注入、组件复用。
- 校园二手交易平台：页面、搜索、表单、Django 模板、数据流。
- AI 实习：模型工作流如何给前端提供结构化数据。

### 手撕练习

推荐题型：

- 最长无重复子串。
- 合并区间。
- 二叉树层序遍历。
- 反转链表。
- LRU Cache。
- 前 K 个高频元素。

### 项目输出

每个项目准备：

- 30 秒版本。
- 3 分钟版本。
- 3 个技术追问。
- 3 个 Bad Case。
- 2 个重做优化方向。

### 自测问题

- 你最有技术含量的前端项目是哪一个？
- 这个项目最难的状态管理问题是什么？
- 你如何证明自己不只是写页面？
- 如果项目上线后用户反馈卡顿，你怎么定位？
